# NB-NTM
Code for 2020 ACL paper ["Neural Mixed Counting Models for Dispersed Topic Discovery"](https://www.aclweb.org/anthology/2020.acl-main.548/)



### Environment Setting

```
Python=3.7.3
Pytorch=1.1.0
CUDA=11.0
```



### Run the Code

Change directory to this folder and run: 

```python
python main.py
```
